# 문제
```
N-Queen 문제는 크기가 N × N인 체스판 위에 퀸 N개를 서로 공격할 수 없게 놓는 문제이다.
N이 주어졌을 때, 퀸을 놓는 방법의 수를 구하는 프로그램을 작성하시오.

- 입력
첫째 줄에 N이 주어진다. (1 ≤ N < 15)

- 출력
첫째 줄에 퀸 N개를 서로 공격할 수 없게 놓는 경우의 수를 출력한다.
```
----

# 풀이 및 구현

문제를 보고 처음 든 생각은 2차원 N × N 체스판에 Queen을 놓으며
그 지점을 기준으로 불가능해지는 위치를 초기화하려 했다.

하지만 이렇게 알고리즘을 설계할 경우, dfs를 들어갈 때마다 여러 칸을 중복 계산하고 복구하는 비용이 발생한다.

시간 복잡도는 무려 $O(N! \times N^2)$이다..

따라서 1차원으로 문제를 바라봐야 했다.

**Queen은 각 행에 하나만 존재**할 수 있기 때문에 1차원으로 처리함에 근거가 된다.

각 행<code>line[y]</code>에서 i번째에 Queen을 놓는게 가능한지(<code>isAble</code>)를 확인하며 가능할때 다음 <code>dfs(y+1)</code>로 넘어간다.

<code>isAble</code>에서 고려해야할 조건은 다음과 같다.
1. 각 열에 하나의 Queen만 존재해야 한다.
2. Queen을 한 점으로 갖는 기울기가 -1이나 1인 직선에는 자신만 존재해야 한다.

아래는 <code>isAble</code>가 false임을 판별하는 조건식이다.
```c++
line[y] == line[x] || abs(x - y) == abs(line[x] - line[y])
```
----

# 코드
```c++
#include <iostream>
#include <cmath>
using namespace std;

int n, ans = 0;
bool isAble;
int line[16];

void dfs(int y) {
    if (n == y) ans++;
    else {
        for (int i = 0; i < n; i++) {
            line[y] = i;
            isAble = true;
            for (int x = 0; x < y; x++) {
                if (line[y] == line[x] || abs(x - y) == abs(line[x] - line[y])){
                    isAble = false;
                    break;
                }
            }
            if (isAble) {
                dfs(y + 1);
            }
        }
    }
}


int main()
{
    cin >> n ;
    dfs(0);
    cout << ans;
}
```
